## 🌸 About

Zeniski is a modern web application that embraces the principles of zen philosophy, bringing tranquility and mindfulness to the digital space. Built with Next.js and TypeScript, this project focuses on creating a peaceful and intuitive user experience.

*The name "Zeniski" combines "Zen" (peaceful state of mind) with a playful twist, representing the harmony between simplicity and functionality.*

## ✨ Features

- 🧘 **Zen-Inspired Design**: Clean, minimalist interface promoting focus and calm
- ⚡ **Fast Performance**: Built on Next.js 14+ with App Router for optimal speed
- 📱 **Responsive Design**: Seamless experience across all devices
- 🎨 **Modern UI**: Utilizes the Geist font family for elegant typography
- 🔧 **Developer Friendly**: Hot reload and modern development tools
- ♿ **Accessibility First**: Built with web accessibility standards in mind

## 🛠️ Tech Stack

- **Framework**: [Next.js 14+](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: CSS Modules / Tailwind CSS (update based on your choice)
- **Font**: [Geist](https://vercel.com/font) by Vercel
- **Deployment**: [Vercel Platform](https://vercel.com/)

## 🚀 Getting Started

### Prerequisites

- Node.js 18.17 or later
- npm, yarn, pnpm, or bun package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Demigod1410/zeniski.git
   cd zeniski
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   # or
   bun install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   # or
   bun dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application.

## 📁 Project Structure

```
zeniski/
├── app/                 # App Router directory (Next.js 13+)
│   ├── globals.css     # Global styles
│   ├── layout.tsx      # Root layout component
│   ├── page.tsx        # Home page component
│   └── favicon.ico     # App favicon
├── components/         # Reusable React components
├── public/            # Static assets
├── styles/            # Additional stylesheets
├── types/             # TypeScript type definitions

├── next.config.js     # Next.js configuration
├── package.json       # Project dependencies
```

## 🎨 Customization

The main page can be edited by modifying `app/page.tsx`. The page auto-updates as you edit the file thanks to Next.js hot reload functionality.

### Key Files to Customize:
- `app/page.tsx` - Main homepage component
- `app/layout.tsx` - Root layout and metadata
- `app/globals.css` - Global styles and CSS variables

## 🌐 Deployment

### Deploy on Vercel (Recommended)

The easiest way to deploy your Next.js app is to use the [Vercel Platform](https://vercel.com/new?utm_medium=default-template&filter=next.js&utm_source=create-next-app&utm_campaign=create-next-app-readme) from the creators of Next.js.

1. Push your code to a Git repository (GitHub, GitLab, Bitbucket)
2. Import your repository on Vercel
3. Vercel will automatically detect it's a Next.js project and configure the build settings
4. Your app will be deployed and you'll get a URL

### Alternative Deployment Options

- **Netlify**: Connect your Git repository for automatic deployments
- **Railway**: Simple deployment with database support
- **DigitalOcean App Platform**: Managed platform with custom domains
- **Self-hosted**: Use `npm run build` and deploy the `out` folder
